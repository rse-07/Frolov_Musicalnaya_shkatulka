# Frolov_Musicalnaya_shkatulka
Цель: шкатулка с музыкой
Задачи:
издаавть музыку при открытии крышки.


Ресурсы:

1. ардуино
2. мкетка
3. аккумулятор
4. переключатель
5. датчик растаяния
6. динамик
7. провода

Состояния:
	1. Ждет
    2. Играет музыку
	3. Разряжено

способности:
играть мелодию
измерять растаяние
остановить мелодию
